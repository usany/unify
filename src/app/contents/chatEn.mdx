# Chat <a id="chat" />
<PiazzaScreen />
This page is for chatting with other users.
You can find sharing partners in group chats that are visible to all users. Clicking a user's profile image allows you to visit their profile or start a conversation.
In private chats, clicking the button to the left of the message input box lets you initiate a video or voice call. Both parties must click the call button to connect.
<br />
*Note: Administrators may access private chat content for accident prevention and response.
