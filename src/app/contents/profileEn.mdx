# Profile <a id="profile" />
<ProfileFirst />
This page allows you to view user profiles.
<ProfileSecond />
On your own profile page, you can upload a profile image and change your name and campus. If you are on campus, you can perform a campus location check, which resets daily. A user's campus location status is visible on the user search page.
<ProfileThird />
Password changes and account deletion are also available from your profile.
<ProfileFourth />
Points, followers, and following counts can be clicked to view detailed information. From another user's profile, you can follow them or navigate to a chat page with them. If you wish to report another user, you can click "Report" to go to the reporting page.
<ProfileFifth />
A graph of the user's sharing frequency and their shared cards are displayed. You can swipe through cards one by one. You can also filter to see only "Borrowing" or "Lending" cards.
